【Task：モデル会話（番号付き・話者対応）】

7. You: Nice. Want to come for a run sometime?
8. Friend: Sure, why not?

==================================================
【Pronunciation & Intonation Focus（Youのセリフのみ）】
==================================================
1. Lately I've started going for a run before work.
- Lately /ˈleɪtli/：最初にアクセント。
- I've /aɪv/：短縮形を自然に。
- started /ˈstɑːrtɪd/：最初にアクセント。
- before /bɪˈfɔːr/：後半にアクセント。

3. I use a fitness app to track my runs – it keeps me motivated.
- fitness /ˈfɪtnəs/：最初にアクセント。
- track /træk/：語尾 /k/。
- runs /rʌnz/：語尾 /z/。
- motivated /ˈmoʊtɪveɪtɪd/：最初にアクセント。

5. How do you stay active?
- stay /steɪ/：二重母音を明確に。
- active /ˈæktɪv/：最初にアクセント。
- do you：自然につなげる。
- 疑問文は上昇調。

7. Nice. Want to come for a run sometime?
- Nice /naɪs/：明るく。
- Want to：自然につなげる。
- run /rʌn/：短くはっきり。
- sometime /ˈsʌmtaɪm/：最初にアクセント。
