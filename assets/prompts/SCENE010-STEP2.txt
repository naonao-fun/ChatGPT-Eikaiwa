==================================================
【Task：モデル会話（番号付き・話者対応）】
==================================================
1) Traveler: Hello. I’d like to buy a SIM card, please.
2) Staff: Sure. We have two plans — the Best USA plan and the Aloha plan.
3) Traveler: What’s the difference between them?
4) Staff: The Best USA plan is cheaper but has less data. The Aloha plan includes more data and lasts for two weeks.
5) Traveler: I’m only here for a few days, so a short-term plan should be fine.
6) Staff: Then the Best USA plan should be perfect for you.
7) Traveler: Great. Could you help me set it up on my phone?
8) Staff: Of course. First you have to download the app.

==================================================
【Pronunciation & Intonation Focus（学習者セリフのみ）】
==================================================

●1) Hello. I’d like to buy a SIM card, please.
- I’d /aɪd/：アイド（wouldの弱形）
- like to /laɪk tə/：トゥは弱く「tə」
- SIM card：SIMをはっきり、cardの/r/を明確に
- 文末pleaseは丁寧にやや下げ調子

●3) What’s the difference between them?
- What’s /wʌts/：ワッツ（tをはっきり）
- difference /ˈdɪfərəns/：ディファレンス（最初に強勢）
- between /bɪˈtwiːn/：トゥイーンを伸ばす
- them /ðəm/：ðを舌先で軽く

●5) I’m only here for a few days, so a short-term plan should be fine.
- I’m /aɪm/：アイム
- only /ˈoʊnli/：オウに強勢
- few /fjuː/：フュー（y音）
- short-term：shortを強め、termの/r/を明確に

●7) Great. Could you help me set it up on my phone?
- Could you /kʊdʒu/：クッジュ（連結）
- help me /help mi/：pを軽く閉じる
- set it up /seɾɪɾʌp/：セリラップ（tのフラップ）
- phone /foʊn/：フォウン（oを伸ばす）
